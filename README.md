# Luma Specialty Coffee — Teruel

Web de una sola página para Luma Specialty Coffee (Ctra. de Alcañiz, 39 · Local 2 · 44003 Teruel).

Hero en vídeo, carta real por secciones, galería con fotos reales, paleta azul marino + caoba.
Abrir `index.html` en el navegador.

## Tipografías de marca

Titulares en **PP Editorial New**; etiquetas y cifras en **00 Hypertext Display Bold**. Logo oficial vectorial (`assets/logo-luma-*.png`).

> ⚠️ **PENDIENTE — licencia de fuentes:** preguntar al local (Luma) si tiene el **permiso/licencia comercial de uso web** de estas tipografías, en especial **00 Hypertext Display** (el archivo dice *NonCommercial*) y **PP Editorial New**. No publicar en producción hasta confirmarlo; si no hay licencia, sustituir por equivalentes libres.
